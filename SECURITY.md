# Security Policy

## Supported versions
Security updates are provided for the latest stable release branch.

## Reporting a vulnerability
Please do not open public issues for security reports.

Report vulnerabilities privately by email to:
- `security@unoforge.dev`

Include:
- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested mitigation (if available)

## Response process
- Initial acknowledgment target: within 72 hours
- Triage and impact assessment: as soon as possible
- Fix timeline: based on severity and exploitability

We will coordinate disclosure after a fix is available.
